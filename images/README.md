# Image Sources

This folder is intended to contain public‑domain copies of Albrecht Dürer's witch engravings. Because the images themselves are large binary files, we provide links to high‑resolution versions hosted on Wikimedia Commons.

## The Four Witches (1497)

- **Wikimedia Commons**: [Albrecht Dürer - The Four Witches](https://commons.wikimedia.org/wiki/File:Albrecht_D%C3%BCrer_-_The_Four_Witches_(Nuremberg_Chronicle).jpg)
- **Description**: Engraving (Bartsch 75). Four female nudes, three facing the viewer and one turned away, standing in a confined space.
- **Public domain**: Yes (artist died in 1528).

## Witch Riding Backwards on a Goat (c. 1500)

- **Wikimedia Commons**: [Albrecht Dürer - Witch Riding Backwards on a Goat](https://commons.wikimedia.org/wiki/File:Albrecht_D%C3%BCrer_-_Witch_Riding_Backwards_on_a_Goat.jpg)
- **Description**: Engraving (Bartsch 67). A naked witch sitting backwards on a goat, holding a distaff and spindle, accompanied by a putto.
- **Public domain**: Yes (artist died in 1528).

## Usage

For local analysis, you may download the images from the links above and place them in this directory, renaming them as:

- `four_witches.jpg`
- `witch_riding_goat.jpg`

The markdown analyses refer to these filenames.
